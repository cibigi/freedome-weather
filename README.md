# freedome-weather
Candidatura front-end developer per Freedome.

Semplice applicativo web per la visualizzazione delle condizioni meteo.

## Tecnologie utilizzate
* HTML
* CSS
* JavaScript
* Open-Meteo API
* Google Material Icons

## Struttura
La pagina è strutturata in tre sezioni rettangolari di uguale misura.
* La prima mostra le condizioni metereologiche attuali per la località preimpostata.
* La seconda le condizioni
* La terza contiene le previsioni per i giorni seguenti.

Le informazioni contenute nella seconda e nella terza sezione sono mostrate in un carosello (cinque elementi alla volta) che è possibile navigare utilizzando i puntini presenti nella parte bassa della card.

## Fetch
Le informazioni vengono ottenute tramite l'API gratuita di Open-Meteo.
